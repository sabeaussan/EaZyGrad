Tensor Object
=============

The public tensor API is implemented by :class:`eazygrad._tensor._Tensor`.
Most users construct tensors through :func:`eazygrad.tensor` or
:func:`eazygrad.from_numpy`, then interact with the returned object through
the methods documented here.
